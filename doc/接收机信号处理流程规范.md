# 5G NR 接收机信号处理流程规范

本文档从 3GPP TS 38.211 / 38.101-1 / 38.213 / 38.215 规范中提取，按接收机信号处理流程顺序组织，涵盖从天线接收到 PBCH 解调的完整物理层处理链。

---

## 1. 小区搜索流程概述

UE 初始接入时执行小区搜索（Cell Search），按以下顺序进行：

1. **频域盲搜 PSS**：在同步栅格候选频点上检测 PSS
2. **时域同步**：通过 PSS 相关检测获取符号定时
3. **频偏估计与补偿**：估计并补偿载波频偏
4. **SSS 检测**：在 PSS 确定的位置检测 SSS，获取完整物理层小区 ID
5. **PBCH 解调**：解调 PBCH 获取 MIB 和系统信息

## 3. 物理层小区 ID

> 规范来源：TS 38.211 Clause 7.4.2.1

```
N_ID^cell = 3 × N_ID^1 + N_ID^2

N_ID^1 ∈ {0, 1, ..., 335}  → 336 个值，由 SSS 检测得到
N_ID^2 ∈ {0, 1, 2}         → 3 个值，由 PSS 检测得到
总计: 336 × 3 = 1008 个唯一物理层小区 ID
```

## 4. PSS（主同步信号）

> 规范来源：TS 38.211 Clause 7.4.2.2

### 4.1 序列生成 (Clause 7.4.2.2.1)

PSS 基于长度 127 的 m 序列，生成多项式 x^7 + x^4 + 1：

```
d_PSS(n) = 1 - 2 × x[(n + 43 × N_ID^2) mod 127],  n = 0, 1, ..., 126

m 序列递推: x(i+7) = [x(i+4) + x(i)] mod 2
初始状态:   [x(6), x(5), x(4), x(3), x(2), x(1), x(0)] = [1, 1, 1, 0, 1, 1, 0]
```

- 乘数 43 使不同 N_ID^2 在同一 m 序列上循环移位，保证正交性
- BPSK 调制: 0 → +1, 1 → -1

### 4.2 PSS 资源映射

> 规范来源：TS 38.211 Clause 7.4.3.1.1

- 映射到 SS/PBCH block 的 **第 0 个 OFDM 符号**
- 占用子载波 **56 ~ 182**（共 127 个）

---

## 5. SSS（辅同步信号）

> 规范来源：TS 38.211 Clause 7.4.2.3

### 5.1 序列生成 (Clause 7.4.2.3.1)

SSS 由两个 m 序列的 Gold 码组合生成：

```
d_SSS(n) = 1 - 2 × [x_0((n + m_0) mod 127) + x_1((n + m_1) mod 127)] mod 2

m_0 = 15 × (N_ID^1/3) mod 127
m_1 = 15 × (N_ID^1/3 + floor(N_ID^1/3)) mod 127

x_0 递推: x_0(i+7) = [x_0(i+4) + x_0(i)] mod 2
x_1 递推: x_1(i+7) = [x_1(i+1) + x_1(i)] mod 2

x_0 初始: [1, 0, 0, 0, 0, 0, 0]
x_1 初始: [1, 0, 0, 0, 0, 0, 0]
```

### 5.2 SSS 资源映射

- 映射到 SS/PBCH block 的 **第 2 个 OFDM 符号**
- 占用子载波 **56 ~ 182**（共 127 个）

## 6. SS/PBCH Block 时频结构

> 规范来源：TS 38.211 Clause 7.4.3.1

### 6.1 总体结构

- **时域**：4 个 OFDM 符号（编号 0 ~ 3）
- **频域**：240 个连续子载波（编号 0 ~ 239）

### 6.2 资源映射表 (Table 7.4.3.1-1)

| 信道/信号 | OFDM 符号 l | 子载波 k |
|----------|------------|---------|
| PSS | 0 | 56, 57, …, 182 |
| SSS | 2 | 56, 57, …, 182 |
| PBCH | 1, 3 | 0, 1, …, 239 |
| PBCH | 2 | 0, 1, …, 47, 192, 193, …, 239 |
| DM-RS (PBCH) | 1, 3 | 每 4 个子载波间隔 |
| DM-RS (PBCH) | 2 | 每 4 个子载波间隔 (与 PBCH 交错) |

```
符号 0: PSS (中央 127 子载波)
符号 1: PBCH + DM-RS (全 240 子载波)
符号 2: SSS + PBCH(部分) + DM-RS
符号 3: PBCH + DM-RS (全 240 子载波)
```

## 7. OFDM 基带信号生成

> 规范来源：TS 38.211 Clause 5.3.1

### 7.1 基本时间单位

```
T_c = 1 / (480 × 10^3 × 4096) ≈ 0.509 ns
T_s = 64 × T_c  (LTE 兼容)
```

### 7.2 OFDM 时域信号

```
s_l^(p,μ)(t) = Σ_k a_{k,l}^(p,μ) × exp(j × 2π × (k + k_0) × Δf × (t - N_{CP,l}^μ × T_c))
               / sqrt(N_{MB}^μ × K)

参数:
  μ      = numerology 索引 (0~6)
  Δf     = 15 × 10^3 × 2^μ Hz  (子载波间隔)
  N      = 2048 × 2^μ           (FFT 点数)
  N_CP,l = 符号 l 的 CP 长度
  k_0    = (N_{MB}^μ - 1) / 2
```

### 7.3 OFDM 参数表

> 规范来源：TS 38.211 Table 4.2-1, Table 4.3.2-1

| μ | SCS (kHz) | FFT 点数 | 每时隙符号数 | 每子帧时隙数 |
|---|----------|---------|------------|------------|
| 0 | 15 | 2048 | 14 | 1 |
| 1 | 30 | 4096 | 14 | 2 |
| 2 | 60 | 8192 | 14 | 4 |
| 3 | 120 | 16384 | 14 | 8 |
| 4 | 240 | 32768 | 14 | 16 |

### 7.4 CP 长度计算

基于基准 FFT = 2048（μ=0, SCS=15kHz）：

| 参数 | 值 (μ=0) | 值 (μ=1) | 公式 |
|------|---------|---------|------|
| 短 CP (l>0) | 144 | 288 | 144 × 2^μ |
| 长 CP (l=0) | 160 | 320 | 160 × 2^μ |

**缩放到任意 FFT 点数**: `CP = 基准CP × (N_FFT / 2048)`

## 8. DM-RS（解调参考信号）

> 规范来源：TS 38.211 Clause 7.4.1.4

### 8.1 PBCH DM-RS 序列

```
r(m) = 1/√2 × (1 - 2c(2m)) + j × 1/√2 × (1 - 2c(2m+1))

伪随机序列初始化:
c_init = 2^11 × (i_SSB + 1) × (2 × N_ID^cell + 1) + N_ID^cell
```

### 8.2 DM-RS 映射

- 每 4 个子载波间隔映射一个 DM-RS 符号
- 用于 PBCH 信道估计

## 9. 频偏估计与补偿

### 9.1 发射端调制上变频

> 规范来源：TS 38.211 Clause 5.4

```
发射信号 = s(t) × exp(j × 2π × f_0 × t)

其中 f_0 为载波频率
```

### 9.2 接收端频偏补偿实现

在 PSS 相关检测过程中：

```python
# 对每个候选 GSCN 频点
for gscn in candidate_list:
    freq_offset = gscn_to_frequency(gscn) - center_frequency
    rx_compensated = rx_signal × exp(-j × 2π × freq_offset × t)

    # 互相关检测 PSS
    correlation = correlate(rx_compensated, pss_template)
```

### 9.3 分段相关抗频偏

在实际工程中，为应对大频偏场景，采用分段相关（Partial Correlation）算法：

1. 将 PSS 序列分为 K 段
2. 各段独立执行互相关
3. 将各段相关值相位补偿后叠加
4. 有效缩短相干积分时间，提升抗频偏能力

## 11. QCL 与信道估计

> 规范来源：TS 38.211 Clause 4.4.1

### 11.1 天线端口

天线端口定义：在该端口上传送符号的信道，可以从同端口另一个符号的信道推断。

### 11.2 QCL（准共址）

UE 可假设同一 SS/PBCH block 内：
- DM-RS 与 PBCH 的信道具有 QCL 关系
- 相同 block 索引的 SS/PBCH block 在以下特性上准共址：
  - 时延扩展、多普勒扩展、多普勒偏移、平均增益、平均时延
